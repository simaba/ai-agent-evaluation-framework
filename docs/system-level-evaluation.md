# System-Level Evaluation

System-level evaluation looks beyond single outputs and measures whether the full agent workflow performs well as an operating system.

## Questions to ask
- Does the full workflow achieve the intended outcome?
- Are the trade-offs between correctness, consistency, latency, and cost acceptable?
- Does the system degrade gracefully when a step fails?
- Can one design be compared fairly against another?

## Design rule
Evaluate the whole workflow, not just the best-looking individual output.
